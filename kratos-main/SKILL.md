# Kratos Framework Skills

This is the central router for all skills related to the Kratos microservices framework. Choose the appropriate sub-skill based on your task.

| **kratos-openapi** | Generating OpenAPI/Swagger documentation. |

| **kratos-docker** | Building Docker images for Kratos services. |
